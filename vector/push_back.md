# push_back

**Difficulty** : Beginner 

**Time Considerations** : 15 minutes

**Description** : Add new elements to the end of a vector

**Syntax :** vectorname.push_back()

**Example**:
```cpp
    //Declare the vector  
    std::vector<int> vector1;

    //Function to push_back values to the vector
    vector1.push_back(1);
    vector1.push_back(2);

```
